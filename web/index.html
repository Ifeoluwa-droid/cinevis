What am I looking at?

This plot shows financial and rating data about recent films. Each circle on the plot represents a film. Each axis can be set to represent a different data field using the menus on the right. The data field used to color the circles can also be set using the menus.
Filters for most of the data fields are also shown at the right. Adjusting the filters changes the data shown on the screen. As data is limited the visible circles will automatically be zoomed in to show only the films that match the current filters.
Hovering over a circle shows details of the film that the circle represents below the plot.

Data

The data includes most films from 2008-2011. The data includes the following fields:
  • Profitability : Percent of budget recovered
  • Budget : Cost, in millions US Dollars
  • Opening Weekend Theaters : Number of Theaters in US Opening Weekend
  • Opening Weekend Revenue : Revenue, in millions US Dollars
  • Opening Weekend per Cinema : Average per cinema on opening weekend, in US Dollars
  • Worldwide Gross : Total revenue, in millions US Dollars
  • Domestic Gross : US revenue, in millions US Dollars
  • Foreign Gross : non-US revenue, in millions US Dollars
  • Audience Rating : Average audience score from Rotten Tomatoes, score: 1-100
  • Critic Rating : Average meta score of critical reviews from Rotten Tomatoes, score: 1-100
  • Oscar : Names of Academy Award categories
  • Studio : Name of the lead studio
  • Genre : Name of the genre from IMDB
  • Story : Name of the story type
The data set is from the in the Information is Beautiful 'Hollywood Budgets' Challenge. The data was pruned to remove films marked 'exclude'. The sources for budget data is TheNumbers.com and BoxOfficeMojo.com.
